## Project uses Laravel Queue.

## To set-up laravel queue follow this steps

php artisan queue:table - created table for jobs

php artisan queue:work - run this to process saved jobs

Also in .env specify new queue_connection

QUEUE_CONNECTION=database # or redis

To use this on production either run queue:work using forever (preferable way)

forever run php artisan queue:work

Or create a cron schedule to run queue:work every minute
